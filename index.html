    Java tool to help create permission files for your Bukkit Minecraft server.

    Project maintained by OxideDevelopment Hosted on GitHub Pages — Theme by mattgraham

    Welcome to PermBuilder.

    PermBuilder is an application written in Java to help server owners easily create and manage permission files for their Bukkit servers. PermBuilder uses SWT for it's GUI, making it have a clean and nice GUI.

    Features

    Since our application is still in development, these are not the permanent features, as they will expand.

    • Automatic update checker
    • Open permission files on your computer, or remotely on a FTP server(This is in development at the moment)
